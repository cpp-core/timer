Timer API¶

template<class Duration = std::chrono::nanoseconds, class Clock = std::chrono::high_resolution_clock>
class Timer¶

The Timer class template implements a timer designed for minimal overhead, ad-hoc timing of code regions including micro-timing down to single machine instructions.
